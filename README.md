 Project Overview

This project focuses on transforming raw Amazon retail sales data (100 transactions) into a dynamic and interactive Excel dashboard. The objective was to clean, structure,
and analyze the dataset using Pivot Tables and XLOOKUP to generate meaningful business insights and support data-driven decision-making.

 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables (Rows, Columns, Values, Filters)
* XLOOKUP for cross-sheet referencing
* IF, IFERROR, SUM, COUNT, AVERAGE
* Date grouping (Month-wise analysis)
* Calculated Fields in Pivot
* Conditional Formatting
* Slicers & Pivot Charts
* KPI Card Creation

Dataset Details

* 100 Sales Records
* Multiple Regions
* Product Categories
* Customer Profiles
* Payment Methods
* Order Date, Delivery Date, Cancel Date
* Delivery Status

 Data Cleaning & Preparation

* Standardized Region, Payment Method, and Delivery Status
* Converted date columns into proper date format
* Removed blanks and extra spaces
* Tagged cancelled orders using Cancel Date and Delivery Status
* Created Delivery Performance column (Fast / Slow)
